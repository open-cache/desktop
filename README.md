# OpenCache Desktop

Native desktop app for [OpenCache](https://www.opencache.ai) — sign in, connect your Nodus SDK key, and your machine becomes a live desktop runtime automatically. No extra authorization steps.

## Download

Grab the latest installer from [Releases](https://github.com/open-cache/desktop/releases):

- **macOS** (Apple Silicon): `OpenCache-<version>-arm64.dmg`
- **Windows** (x64): `OpenCache Setup <version>.exe`

> Preview builds are unsigned — on macOS, right-click → Open on first launch; on Windows, choose "More info → Run anyway" in SmartScreen.
